# Computational-Photography
Includes plans of implementing autofocus, auto white balance, and other computational photography methods. Uses knowledge from Digital Camera and Computer Vision Lab at National Taiwan University to implement custom algorithms for usage for digital cameras and development of image processing pipelines for AI training.
